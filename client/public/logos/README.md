# Logos des Partenaires

## Instructions

Placez les logos de vos partenaires dans ce dossier.

### Format recommandé

- **Type de fichier** : PNG ou SVG (avec fond transparent de préférence)
- **Dimensions** : 200x100 px (largeur x hauteur) ou ratio similaire
- **Poids** : < 100 KB par logo
- **Couleur** : Idéalement en noir/gris ou version monochrome

### Nommage des fichiers

Utilisez des noms descriptifs en minuscules :
- `pennylane.png`
- `expert-comptable-paris.png`
- `ordre-experts-comptables.svg`
- etc.

### Exemple de structure

```
public/logos/
├── pennylane.png
├── sage.png
├── quickbooks.png
├── ordre-experts-comptables.png
└── ...
```

## Le composant détectera automatiquement tous les logos

Dès que vous ajoutez un logo dans ce dossier, il apparaîtra automatiquement dans le défilement sur la page d'accueil.
